# Epic 110: SpatialRust Vision 1.0 gate

Epic 110 closes the OpenCV-competitive roadmap with a fail-closed release
policy. Ten named conformance/audit cases, seven OpenCV receipts, two runnable
examples, the four-item security baseline, fixed performance measurements, and
the `vision-1` migration policy are mandatory. Missing, skipped, or over-budget
evidence returns all denial reasons instead of silently passing.

Canonical ceilings are 50 ms for VGA CPU, 250 ms for 1080p CPU, 20 ms for the
resident 4K GPU chain, and 64 MiB for explicit copies in one graph run. The
recorded Epic 104 4K GPU chain value is 13.523 ms, below its release ceiling.

The final local candidate passed 92 vision unit tests, 10 property tests, the
stable facade integration test, 44 GPU all-feature tests, the Python suite, and
all seven OpenCV aggregate suites (calibration, correctness, performance,
RGB-D, video, odometry, and photography).
