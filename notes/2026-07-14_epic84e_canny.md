## Contract

The `imgproc-canny` feature provides edge-only and inspectable-result APIs. It
accepts u8 single-channel packed or strided views, validates finite non-negative
thresholds and Sobel aperture 3, 5, or 7, supports L1 and L2 magnitude, and
returns an edge map containing only 0 or 255. Gradient images, magnitude, and
the directionally suppressed response are available without changing the
edge-only API.

The implementation uses explicit replicated borders, integer-compatible
non-maximum suppression, and 8-neighbor hysteresis. Aperture 7 follows the
OpenCV 1/16 Sobel scale and half-to-even `CV_16S` conversion contract.
